# In-App Help Guide Portal

The Route Builder IDE integrates a Windows-style **Help Guide** directly into the workspace to provide immediate, context-aware assistance.

---

## 1. Interface Layout

The Help Guide window is divided into three key panels:
- **Search Sidebar**: A filter field to search titles, keywords, and content across all help files dynamically.
- **Categorized Index Tree**: Nested category folders (Overview, Route Designer, Kamelet Studio, Dependency Catalog, Validation Studio, Advanced Tools) for structured browsing.
- **Rich Document Viewer**: A dark-themed HTML WebView rendering lists, blockquotes, syntax highlights, and diagrams.

---

## 2. Context-Aware Pre-Filtering

Every major tool window in the IDE features a dedicated **Help** button. Clicking the button opens the Help Guide pre-loaded or pre-filtered to the corresponding topic:

- **Validation Studio**: Filters to the "Validation Studio" category.
- **Kamelet Studio**: Filters to the "Kamelet Studio" category.
- **Dependency Catalog**: Filters to the "Dependency Catalog" category.
- **Crypto Studio**: Searches for "Crypto".
- **XSLT Mapper**: Searches for "XSLT".
- **Faker & Mock Studio**: Searches for "Faker".
